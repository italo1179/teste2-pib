Configurar Planilha de Eventos

Siga os passos abaixo uma única vez. Depois disso, a pessoa responsável pelos eventos só precisa abrir a planilha no Google e editar normalmente — sem tocar em nenhum código.

Atenção: Esta página é só um guia. Não tente abrir ou enviar este arquivo para o Google Drive — isso não cria a planilha.
Siga os passos abaixo e tudo vai funcionar corretamente.

Passo a passo
1
Crie uma planilha em branco no Google Sheets Clique no botão abaixo — ele abre uma planilha nova automaticamente. Não tente abrir o arquivo desta página no Google Drive, isso não funciona. Abrir nova planilha no Google Sheets
2
Abra o editor de script No menu superior da planilha, vá em Extensões → Apps Script. Uma nova aba vai abrir.
3
Apague o código existente e cole o novo Selecione tudo que está no editor (Ctrl + A), delete, e cole o código abaixo usando o botão "Copiar Script".
4
Clique em ▶ Executar Clique no botão de play (▶) no topo do editor. Na primeira vez, o Google vai pedir permissão — clique em Permitir. A planilha será criada automaticamente.
5
Publique a planilha na web De volta na planilha, vá em Arquivo → Compartilhar → Publicar na web → selecione a aba Eventos PIB Rio Verde → formato CSV → clique em Publicar.
6
Copie o ID da planilha e me envie Olhe a URL da planilha: docs.google.com/spreadsheets/d/ID_AQUI/edit. Copie o trecho longo entre /d/ e /edit e me manda — eu coloco no site.
Script para copiar
📋  configurarPlanilha.gs 
function configurarPlanilha() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getActiveSheet();

  sheet.setName("Eventos PIB Rio Verde");
  sheet.clear();
  sheet.clearFormats();

  // Largura das colunas
  sheet.setColumnWidth(1, 65);   // A: Dia
  sheet.setColumnWidth(2, 82);   // B: Mes
  sheet.setColumnWidth(3, 300);  // C: Titulo
  sheet.setColumnWidth(4, 360);  // D: Detalhes
  sheet.setColumnWidth(5, 220);  // E: Link
  sheet.setColumnWidth(6, 160);  // F: Texto do link

  // Linha 1 - Titulo principal
  sheet.setRowHeight(1, 58);
  const r1 = sheet.getRange("A1:F1");
  r1.merge();
  r1.setValue("EVENTOS - PIB RIO VERDE");
  r1.setBackground("#0d0050");
  r1.setFontColor("#eef2ff");
  r1.setFontSize(17);
  r1.setFontWeight("bold");
  r1.setHorizontalAlignment("center");
  r1.setVerticalAlignment("middle");

  // Linha 2 - Aviso de instrucao
  sheet.setRowHeight(2, 30);
  const r2 = sheet.getRange("A2:F2");
  r2.merge();
  r2.setValue("Preencha apenas a partir da linha 6. Nao altere as linhas 1 a 5.");
  r2.setBackground("#1a1a6e");
  r2.setFontColor("#c5cae9");
  r2.setFontSize(11);
  r2.setFontStyle("italic");
  r2.setHorizontalAlignment("center");
  r2.setVerticalAlignment("middle");

  // Linha 3 - Cabecalho das colunas
  sheet.setRowHeight(3, 38);
  const r3 = sheet.getRange("A3:F3");
  r3.setValues([["DIA", "MES", "NOME DO EVENTO", "DETALHES (horario e local)", "LINK (opcional)", "TEXTO DO LINK"]]);
  r3.setBackground("#3949ab");
  r3.setFontColor("#ffffff");
  r3.setFontWeight("bold");
  r3.setFontSize(12);
  r3.setHorizontalAlignment("center");
  r3.setVerticalAlignment("middle");
  r3.setBorder(true, true, true, true, true, true,
    "#5c6bc0", SpreadsheetApp.BorderStyle.SOLID);

  // Linha 4 - Descricao de cada coluna
  sheet.setRowHeight(4, 44);
  const r4 = sheet.getRange("A4:F4");
  r4.setValues([[
    "Numero\ndo dia",
    "Abreviacao\ndo mes",
    "Nome oficial do evento como sera divulgado",
    "Horario, local e detalhes importantes",
    "URL de inscricao ou informacoes\n(deixe vazio se nao tiver)",
    "Texto do botao\nEx: Saiba mais"
  ]]);
  r4.setBackground("#5c6bc0");
  r4.setFontColor("#e8eaf6");
  r4.setFontSize(10);
  r4.setFontStyle("italic");
  r4.setHorizontalAlignment("center");
  r4.setVerticalAlignment("middle");
  r4.setWrap(true);

  // Linha 5 - Linha de exemplo
  sheet.setRowHeight(5, 50);
  const r5 = sheet.getRange("A5:F5");
  r5.setValues([["22", "Jul", "Culto de Aniversario da Igreja",
    "Domingo · 19h00\nAuditorio Principal", "", ""]]);
  r5.setBackground("#e8eaf6");
  r5.setFontColor("#1a237e");
  r5.setFontWeight("bold");
  r5.setFontSize(11);
  r5.setVerticalAlignment("middle");
  sheet.getRange("A5").setHorizontalAlignment("center");
  sheet.getRange("B5").setHorizontalAlignment("center");
  sheet.getRange("D5").setWrap(true);
  sheet.getRange("C5").setNote(
    "Esta e uma linha de exemplo. Pode editar ou excluir."
  );

  // Linhas 6 a 30 - Area de preenchimento
  for (let i = 6; i <= 30; i++) {
    sheet.setRowHeight(i, 48);
    const row = sheet.getRange(i, 1, 1, 6);
    row.setBackground(i % 2 === 0 ? "#f3f4fb" : "#ffffff");
    row.setFontSize(12);
    row.setFontColor("#1a1a2e");
    row.setVerticalAlignment("middle");
    row.setBorder(null, null, true, null, null, null,
      "#dde0f0", SpreadsheetApp.BorderStyle.SOLID);
    sheet.getRange(i, 1).setHorizontalAlignment("center");
    sheet.getRange(i, 2).setHorizontalAlignment("center");
    sheet.getRange(i, 4).setWrap(true);
  }

  // Dropdown de meses na coluna B (linhas 5 a 30)
  const monthRule = SpreadsheetApp.newDataValidation()
    .requireValueInList([
      "Jan","Fev","Mar","Abr","Mai","Jun",
      "Jul","Ago","Set","Out","Nov","Dez"
    ])
    .setAllowInvalid(false)
    .setHelpText("Selecione o mes do evento na lista")
    .build();
  sheet.getRange("B5:B30").setDataValidation(monthRule);

  // Travar as 4 primeiras linhas na tela
  sheet.setFrozenRows(4);

  // Protecao do cabecalho (linhas 1 a 4) - aviso ao tentar editar
  const prot = sheet.getRange("A1:F4").protect();
  prot.setDescription("Cabecalho - nao editar");
  prot.setWarningOnly(true);

  // Protecao da linha de exemplo - aviso ao tentar editar
  const protEx = sheet.getRange("A5:F5").protect();
  protEx.setDescription("Linha de exemplo - pode apagar quando quiser");
  protEx.setWarningOnly(true);

  SpreadsheetApp.getUi().alert(
    "Planilha configurada com sucesso!\n\n" +
    "Proximo passo - publicar para o site funcionar:\n" +
    "Arquivo > Compartilhar > Publicar na web\n" +
    "Selecione a aba 'Eventos PIB Rio Verde'\n" +
    "Formato: CSV > Publicar\n\n" +
    "Depois copie o ID da URL da planilha\n" +
    "(o trecho longo entre /d/ e /edit)\n" +
    "e me mande para eu colocar no site."
  );
}

Ela preenche: Dia · Mês (dropdown) · Nome do evento · Detalhes · Link (opcional).
O site atualiza automaticamente em poucos segundos.
